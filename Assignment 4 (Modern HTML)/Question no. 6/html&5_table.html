Feature HTML HTML5
1. DOCTYPE Required to specify the version of HTML. Simplified to ' DOCTYPE html '
2. Syntax More strict. More forgiving with optional elements.
3. New Elements Limited number of semantic elements. Introduced many new semantic elements (e.g., header, footer, nav, etc.).
4. Offline Support No built-in support for offline web apps. Introduced Application Cache for creating offline web applications.
5. Canvas and SVG Not supported natively. Introduced 'canvas' for 2D drawing and retained support for Scalable Vector Graphics (SVG).
